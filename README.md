# cccut ver.2.0
                                                by Zhidao
                                                2004.11.23. 作成
                                                2022. 1. 2. 最終更新
--------------------------------------------------------------------
### 【説明】

FlyingShineのWindows用ゲーム「CROSS†CHANNEL」アーカイブファイル.pd
の中身を切り出すツール。

PackOnlyファイル（cg.pdおよびbgm.pd）はそれぞれPNGファイルとOGGファ
イルを連結しただけのものなので、デコードなしで読み取り可能です（PNG
対応イメージビューア、OggVorbis対応プレイヤをそれぞれ別途ご用意下さ
い）。

PackPlusファイル（script.pdおよびse.pd）は切り出し後にデコードが必要
です（デコーダは本ツールには含めていません）。

--------------------------------------------------------------------
### 【コンパイル】

makeすれば実行ファイルcccutが出来ます。

--------------------------------------------------------------------
### 【使い方】

% cccut <.pd file> [file no.]
または
% cccut <.pd file>

file no.は、切り出したいアーカイブ内のファイル番号（0スタート）です。
実際にアーカイブに収められているファイル数以上の数が指定された場合に
はエラーになります。

file no.を省略すると、全てのファイルが切り出されます。
